# 活动状态表模板

在飞书中创建一张多维表格，字段如下：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 活动名称 | 文本 | 唯一标识 |
| 触发人 | 人员 | @发起人 |
| 触发时间 | 日期 | 自动记录 |
| 活动类型 | 单选 | 比赛/分享会/团建/自定义 |
| 主题确认状态 | 单选 | 待执行/执行中/已完成/失败 |
| 主题确认产物 | 链接 | 确认单文档 |
| 设计图状态 | 单选 | 待执行/执行中/已完成/失败 |
| 设计图产物 | 链接 | 白板链接 |
| 采集状态 | 单选 | 待执行/执行中/已完成/失败 |
| 采集产物 | 链接 | 报名表链接 |
| 验证状态 | 单选 | 待执行/执行中/已完成/失败 |
| 验证产物 | 链接 | 初审报告 |
| 策划状态 | 单选 | 待执行/执行中/已完成/失败 |
| 策划产物 | 链接 | 方案文档 |
| 最终报告 | 链接 | 汇总文档 |
| 全局状态 | 单选 | 进行中/已完成/已暂停/失败 |
| 备注 | 多行文本 | 错误日志、备注信息 |

## 创建命令

```bash
lark-cli base +table-create \
  --name "活动大管家-状态表" \
  --fields "活动名称:text,触发人:user,触发时间:date,活动类型:select,主题确认状态:select,主题确认产物:url,设计图状态:select,设计图产物:url,采集状态:select,采集产物:url,验证状态:select,验证产物:url,策划状态:select,策划产物:url,最终报告:url,全局状态:select,备注:text"
```
