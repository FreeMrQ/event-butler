# 活动状态表模板

在飞书中创建一张多维表格，字段如下：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 活动名称 | 文本 | 唯一标识 |
| 发起人 | 人员 | @发起人，首次触发时记录 |
| 发起人 open_id | 文本 | 发起人的 open_id，供下游 Agent 使用 |
| 触发时间 | 日期 | 自动记录 |
| 目标群聊 ID | 文本 | 通知发送目标群 oc_xxx |
| 活动类型 | 单选 | 比赛/分享会/团建/自定义 |
| 报名 share 链接 | 链接 | 发布后的表单分享链接 shrn... |
| 海报路径 | 文本 | 海报文件本地路径 |
| 主题确认状态 | 单选 | 待执行/执行中/已完成/失败 |
| 主题确认产物 | 链接 | 确认单文档 |
| 设计图状态 | 单选 | 待执行/执行中/已完成/失败 |
| 设计图产物 | 链接 | 白板链接 |
| 采集状态 | 单选 | 待执行/执行中/已完成/失败 |
| 采集产物 | 链接 | 报名表 Base 链接 |
| 验证状态 | 单选 | 待执行/执行中/已完成/失败 |
| 验证产物 | 链接 | 初审报告 |
| 策划状态 | 单选 | 待执行/执行中/已完成/失败 |
| 策划产物 | 链接 | 方案文档 |
| 最终报告 | 链接 | 汇总文档 |
| 全局状态 | 单选 | 进行中/已完成/已暂停/失败 |
| 备注 | 多行文本 | 错误日志、备注信息 |

## 创建命令

```bash
lark-cli base +table-create \
  --name "活动大管家-状态表" \
  --fields "活动名称:text,发起人:user,发起人open_id:text,触发时间:date,目标群聊ID:text,活动类型:select,报名share链接:url,海报路径:text,主题确认状态:select,主题确认产物:url,设计图状态:select,设计图产物:url,采集状态:select,采集产物:url,验证状态:select,验证产物:url,策划状态:select,策划产物:url,最终报告:url,全局状态:select,备注:text"
```
